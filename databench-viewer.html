Editor

Open a DataBench project

Open a .databench.json project exported from DataBench, or use the editor’s Export Mobile Viewer option to create a standalone file that opens directly on phones and tablets.

Projects are read locally in your browser. Nothing is uploaded. On a hosted site, the viewer can also load a project with ?project=../path/to/project.databench.json.
